# WiM 24 Abril 2026 — De la Brecha a la Oportunidad

Presentación del panel **IA para Mujeres en Minería** — Women in Mining Chile, 24 de abril de 2026.

## Ver la presentación

Abre `index.html` en cualquier navegador moderno, o visita la URL publicada en GitHub Pages.

**Navegación:** ← / → para avanzar, o click en los bordes.
**Fullscreen:** F11.

## Publicar en GitHub Pages

1. Sube esta carpeta a un repo público
2. Settings → Pages → Branch: `main` → Save
3. URL lista en ~1 minuto
